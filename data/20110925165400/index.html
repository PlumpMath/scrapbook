2011精选软件：Lifehacker Pack for Windows

作者: 善用佳软　　日期: 2011-08-09　　分类: 0 软件清单　　 标签: best-list, translation

Lifehacker是国外知名的生活工作效率类网站。近期发布了2011年推荐的Windows软件清单，并提供打包下载。(via) 为方便读者了解，先做简单整理，再陆续翻译，并欢迎读者翻译后留言提交。

本文目录：效率软件 网络沟通 工具软件 图像影音 安全软件 补充软件

一、效率软件｜Productivity

文本编辑器：Notepad++

办公文档用Office处理，纯文本用什么处理呢？当然要找一款专业的编辑器。Notepad++ 就拥有纯文本高效编辑所需的各种功能：文本搜索、多页签界面、各种代码的语法高亮……内置的脚本语言更可以自主增加所需功能。

需要更简单的替代品？请试用 Notepad2。另有付费软件，比如 SublimeText，似乎增加的东西只有更美观的界面。

[下载页]

文本扩展工具：PhraseExpress

用户键入缩写，文本自动扩展——这是节约时间的必备功能。而 PhraseExpress 就是 Windows 平台的最佳文本扩展软件。它虽不完美，但非常实用（建议：取消内置的自动纠错功能）。建立后自己的扩展词条后，在任何程序中键入少数几个字母，它就能替换为一段话、多行代码、日期时间等信息。

如需更简洁的方案，推荐 Texter，或 用 AutoHotkey 创建自己的文字扩展。

[下载页]

文件辅助管理：Belvedere

belvedere 是 Texter 作者 Adam Pash 的另一作品，是基于 Mac Hazel 的文件自动分类软件。通过按照文件类型移动和分类文件、按创建日期自动删除文件、以及其他规则，你的文件管理工作会更有条理、更有效率。只须一次设定，它便自动运行，不再需要占用你的时间。桌面不再凌乱，文件不再会忘记备份到Dropbox。

[下载页]

PDF阅读：SumatraPDF

老实说，没人喜欢用缓慢的 Adobe Reader 打开 pdf 文档。而 SumatraPDF 刚好与 Adobe Reader 相反：简洁、快速、易用。它在瞬间打开 pdf 文档，支持包括表格、注释在内的绝大多数pdf特性。另外，对于热键爱好者来说，它具备丰富的热键，可以让你飞速操作。

需要更多功能？推荐 PDF-XChange。

[下载页]

二、网络沟通｜Internet/Communication

浏览器：Google Chrome

Chrome 已经 超越 Firefox 成为最受高级用户喜爱的浏览器。原因也很自然：更快、可扩展，并通过Google账号同步使用偏好、扩展、密码等数据。另外，它积极升级，每隔几周就通过小而持续的增量更新取得进步。

[下载页]

邮件管理：Thunderbird

多数用户已经改用 web mail，但我们仍然建议安装一个邮件客户端，用于备份、离线访问。Thunderbird 就是这类软件中的最佳选择，支持所有的 IMAP 邮箱，通过众多扩展可以实现高度个性化。

如果有更多需求，并可以付费，那么 Microsoft Outlook 和基于 Thunderbird 的 Postbox 都可以考虑。

[下载页]

网络聊天：Pidgin

想集中管理多个在线聊天账号？pidgin 就是解决方案。支持 AIM, Google Talk, Yahoo Messenger, MSN, Facebook Chat 以及更多服务，并且所有好友名单集中管理。通过第3方扩展，还可以添加更多IM服务、主题、通知，并与社交网络集成。可谓是在线聊天与社交网络的一站式解决方案。

其他推荐：跨平台的 Trillian，主攻社交网络的 Digsby。

[下载页]

系统工具：Adobe Flash


无论你是否喜欢，很多网站仍然通过 Adobe Flash 播放视频、驱动应用，或只是为网站提供导航。因此，尽管它不是最稳定或最快速的插件，我们仍然建议安装。如果它引起了某些问题，也可以通过 FlashBlock for Chrome 和 FlashBlock for Firefox 在浏览器中禁用 flash。

[下载页]

在线交流：Skype

Skype 仍然是最成功的在线视频交流工具，更不用说它也是最便宜的国际电话提供商。它支持几乎所有平台，包括 Windows, Mac, Linux, iPhone, Android 等。就算不是 Skype 迷，也不妨提前安装，有备无患。或许有一天，有人要通过它与你联系呢。

备选方案：Google Chat、Google+

[下载页]

三、工具软件｜Utilities

最小巧的BT软件：?Torrent

?Torrent 是我们最喜爱的Windows平台 BT 客户端，体积小巧，但功能丰富。除了下载bT，还可以方便与朋友分享文件、web界面远程控制种子、下载的同时查看视频。如果你喜欢它，还可以参见这些 提高速度和安全性的技巧。

【xbeta注】虽少用BT，但推崇此为经典优质软件。入选46款最佳免费软件，并专文介绍。

[下载页]

Usenet工具：SABnzbd

不喜欢BT者，可以尝试 用 Usenet 分享和下载文件。而 SABnzbd 则是我们推荐的 Usenet 客户端。它易于搭建、后台运行、通过web可在任意桌面上控制。它的操作步骤高度自动化，因此，你只要载入 NZB 文件，它就会完成后续工作。

【xbeta注】Usenet看起来很好，但在资源和实用性方面（尤其在国内）与BT尚有很大差距。参见 维基百科 和 阮一峰 的介绍。

[下载页]

最经典的同步工具：Dropbox

【xbeta注】景德镇之外的工具，不译。

Dropbox is still the best file syncing tool around, letting you back up files to the web and sync them between all of your machines and mobile devices. You get 2GB of space to start, but it's easy to get more space for free, and useful for so much more than just syncing your documents. It's a must have on any system you own.

If you don't like Dropbox or have problems with it's recent security issues, there are a lot of alternatives out there to consider.

[下载页]

灾备软件：Crashplan

PC用户当备份其系统，而 Crashplan 正是 我们最喜爱的备份工具。不仅可以备份到本地硬盘、远程PC，还可以付少许费用，备份到 Crashplan 的云端，以备意外灾难。如果你不想付费，也可以与朋友的电脑互相备份。

【xbeta注】信息过载时代，硬盘多垃圾。只备份必要的数据即可。

[下载页]

镜像刻录软件：ImgBurn

如果你曾刻录CD和电影，就应该对镜像文件不会陌生。ImgBurn 就是这方面最小巧、易用的刻录工具。几次点击，轻松完工：启动程序、加载ISO、点击刻录。

【xbeta注】较少做刻录工作，但很推荐 ImgBurn。见介绍文章 刻录工具ImgBurn：软件虽小，能耐不小，其中还对其他刻录软件进行了简要点评。

[下载页]

最著名的系统清理软件：CCleaner

过期的临时文件、缓存、系统垃圾…… CCleaner 就是最高效，并且安全的清理工具。支持定期任务，因此无需担心遗忘。Windows 用户必备利器。

【xbeta注】偶用此类软件，但不做深入研究。参小众软件介绍。

[下载页]

软件卸载：Revo Uninstaller

Windows有内置的软件卸载功能，但有些情况下它卸载并不完整（所以才需要CCleaner清理垃圾）。Revo Uninstaller 则会清理软件的每项遗留信息，并能用来管理启动项，以保持系统快速启动。

【xbeta注】不太用这类软件，偶然卸载，就用程序自带快捷方式，或 TC 中的UnInstaller 扩展。

[下载页]

压缩软件：7-Zip

Windows可以内置支持zip格式，但其他压缩格式（7z、rar……）需要外部程序的支持。7-Zip简洁、强大、免费、开源，为最佳选择。

【xbeta注】尽管我主要用TC处理压缩文件，但 7-Zip 还是必装软件。我的口号是 善用7-Zip，代替收费的Winrar和Winzip。

[下载页]

四、图像影音软件｜Music, Pictures, and Video

多媒体播放器：KMPlayer

或许 VLC 是Winodws平台最流行的播放器，但 KMPlayer 则是 我们的最爱。它功能强大、支持格式丰富，甚至能播放不完整的视频。

备选方案：扩展包中提供了 VLC 。其他还有 Media Player Classic, SMPlayer, 和 PotPlayer。

【xbeta注】很少看视频，装的就是 KMPlayer。

[下载页]

数码照片管理：Picasa

如果要管理大量的数码照片，则 Picasa 就是终极方案。只须将照片导入，它就会建立易于浏览的照片库，并提供一些基本但实用的编辑功能。此外，你还可以通过免费的在线相册功能，备份或朋友分享照片。

备选方案：Windows Live Photo Gallery 是Windows集成的照片管理工具。更专业用户或许可以选择 Adobe Lightroom。

【xbeta注】我使用 Picasa 并不多，但认为这才是真正的“数码照片管理工具”。要注意的是：① Picasa适合管理数码照片，而非图片。② IrfanView等是图片查看工具，尽管有强大的批处理功能，但并不是管理工具。所谓管理工具，面对的不是个体，而是很多个体组成的集合。

[下载页]

图像编辑软件：Paint.NET

当 Photoshop 或 GIMP 过于复杂时，Paint.NET 就是简单图像编辑恰到好处的选择。它功能丰富，从裁剪到消除红眼，甚至图层支持。倒是没有你认为多余的其他功能。但对大多数用户而言，功能完全足够。

【xbeta评】Paint.NET 的功能并不象上文说得那么少。而 GIMP 也不象上文说得那么复杂。我主推GIMP，但也认为 Paint.NET 和 Pixia 是好选择。

[下载页]

多媒体管理：iTunes

iTunes 并非我们最偏好的音乐播放器，但它确实无处不在。尤其当你有苹果设备需要管理时，就更离不开它。它具备基本的音乐库管理功能，并且，很可能是多数用户真正需要的软件。

【xbeta】我知道它的音乐管理方式很好。但作为不听音乐的用户，我很少使用 iTunes。另外，我的 iPod Touch 与 iPad 也主要用wifi直接安装程序，很少用 iTunes。我承认，还是不太习惯它。

[下载页]

五、安全软件｜Security

安全套装：Microsoft Security Essentials

Windows 下的安全软件有很多，但并不是只有花钱才能得到足够的安全性。微软的这一产品与Windows集成性好，杀毒率高，不会提醒你付费。并且，相比同类产品，非常轻巧。Microsoft Security Essentials 正是你的安全所需。

【xbeta注】有些用户追求“最安全”的软件，我觉得不必要。多数情况下，好的习惯+基本的安全软件足矣。中国个人用户，似乎多用免费小红伞或 360。

[下载页]

六、更多精选软件｜The Extended Pack

Firefox

Chrome may be our favorite browser, but that doesn't mean Firefox doesn't have anything going for it. Firefox is still more customizable than any other browser you'll find, and it's getting faster all the time. And, with multiple channels, a rapid release cycle, and a new interface on the way, it's catching up to Chrome big time.

[下载页]

LibreOffice

OpenOffice doesn't have the momentum it used to, and the OpenOffice community has left Oracle's offering in favor of LibreOffice, the new standard open-source office suite. It's got a few features OpenOffice doesn't, as well as a more active community. If you don't want to pay for Microsoft's offerings, LibreOffice is the way to go.

[下载页]

Microsoft Office and Office Viewers

Love it or hate it, many of you might need to edit documents in Microsoft Office (or at least view them). Thus, we've added a trial version of Office 2007 Standard to the pack, as well as the Office Viewers for those that only need to open a file here and there.

.NET, Silverlight, and Java

You're probably going to need all three of these frameworks for something. Better to install them all now and get it over with—after all, waiting for Silverlight to download is a pretty big buzzkill when all you want to do is watch Netflix.

Recuva

Accidentally deleting files is a horrifying thing that can happen to anyone. Recuva can often get it back, so it's a good tool to have in your back pocket to help keep you cool in those emergencies.

[下载页]

Teracopy

When you have to copy big files between drives, TeraCopy is a great alternative to Windows Explorer. It'll move them as fast as possible, pause and resume, and even recover from errors.

[下载页]

K-Lite Codec Pack

Windows Media Player is actually a pretty nice player, and if you'd rather use it than install a separate program, this pack of codecs will make sure that it—and any other player—can handle every video format you might run into.

[下载页]

VLC

We understand that many of you are still in love with VLC, and we understand—it might not be our favorite video player, but it's still one of the best and most popular out there. If you can't live without the lightweight, play-anything VLC, we've included it in the extended pack for you.

[下载页]

Foobar2000, MediaMonkey, and Winamp

If you're lucky enough to be free of iTunes (or you only use it to manage your iPod), these three players are some fantastic music managers. Foobar2000 is minimal and customizable, MediaMonkey is an library-organizing monster, and Winamp is just a great all-around player for any user.

Spotify

While the above players are each desktop powerhouses in their own right, we've also fallen in love with the cloud music service Spotify. It's great for discovering new music and testing it out before you buy, so even if you have a local music player, we can't recommend Spotify enough.

[下载页]

Adobe Reader

It's slow, naggy about updates, and somewhat insecure, but unfortunately, sometimes you just need Adobe Reader. Whether you've got PDF files with special features Sumatra doesn't support, or your file just won't open correctly, chances are it'll work in Adobe Reader. We recommend keeping SumatraPDF as your default reader, and keeping Adobe around for those unfortunate occasions.

[下载页]

AutoHotkey

it's one of the geekier selections on our list, so it isn't for everybody, but AutoHotkey is one of the most powerful, useful programs you can get for Windows. With just a bit of code, you can turn just about any action in to a keyboard shortcut, or even create your own little programs. It may look intimidating, but it's surprisingly easy to learn, so if you've ever thought "my life would be perfect if I just had this keyboard shortcut", AutoHotkey is a godsend.

[下载页]

  • Izual_Yang: 2011-08-04 13:43, #28082

    Firefox
    Chrome可能是小编最喜爱的浏览器，但是这并不意味着Firefox完全没有值得一试的优点。Firefox仍然比其他任何浏览器更具有可定制性，而且它的运行速度也在持续改进中。同时，拥有多个更新频道，快速的发布周期，和即将推出的新界面，Firefox正在大步追赶Chrome。

    LibreOffice
    OpenOffice已是强弩之末，而OpenOffice社区也已弃Oracle而转投开源办公套件的新标杆LibreOffice。它拥有一些OpenOffice不具备的新功能，社区也更具活力。如果你不想掏钱给M$，那么LibreOffice是必然的选择。

    Microsoft Office和Office文档查看器
    无论是爱是恨，大多数人还是不得不编辑——至少是查看——微软Office格式的文档。那么我们也只好在列表中加上试用版的Office 2007标准版，以及免费的Office文档查看器——给那些只需随时随地打开文档查看的用户。

    .NET，蓝泽光，爪哇咖啡
    你正在上这个网站，就说明你总有一天会需要这三个玩意。长痛不如短痛还是趁早把它们都装了吧——不然等你一心想在Netflix上看大片的时候，还要等待蓝泽光的下载过程，那真是败兴之极。

    Recuva
    手滑删错文件是每个人身上都可能发生的恐怖事件。Recuva常常都能化险为夷，常备重要哦~

    Teracopy
    如果你需要在不同磁盘之间拷贝大文件，TeraCopy是资源管理器的强大替代品。它能以最快的速度复制文件，支持暂停和恢复，出错后甚至还支持续传。
    （吐槽：另一个TC——Total Commander才是完全替代资源管理器的吧。）

    K-Lite解码包
    Windows Media Player已经是个不错的播放器了（吐槽：真没追求），而且如果你不想再装一个播放器，那么这个解码器合集足以保证WMP——以及其他（吐槽：基于directshow的）播放器——都能够播放你所可能碰到的任何视频格式。

    VLC
    我知道你们中的很多人仍然喜欢VLC，我也知道它可能不是我们最推崇的视频播放器，不过它仍然是最优秀最有人气的之一。所以为了你们这些不用VLC会死星人，在这个扩展列表中我们还是列上了这个轻量级的、啥都能放的利器。

    Foobar2000，MediaMonkey，以及Winamp
    如果你运气好没得iTunes病的话（或者只用iTune来管理iPod），那么请试试这三个神奇的音乐管理器。Foobar2000风格极简，易于定制；MediaMonkey是媒体库管理利器；而Winamp则是功能全面，任何用户都易于上手。

    Spotify
    如果说以上播放器都是在桌面程序中称霸一方的话，那么身处云端的Spotify这个在线音乐服务同样是小编们的最爱。用它来找到新的音乐，在购买前试听，最方便不过了。所以就算您已经安装了本地音乐播放器，我们仍觉得再怎么推荐Spotify都不为过。

    Adobe Reader
    运行巨慢，升级提示巨烦，不太安全——但不幸的是，有时候你非用Adobe Reader不可。无论是碰上带有特殊功能的PDF文件而Sumatra却不支持，或者你用别的阅读器就是死活打不开它，Adobe Reader就是最后那根救命稻草。小编建议把SumatraPDF设为默认阅读器，同时留着Adobe Reader以备不时之需。

    AutoHotkey
    这是本列表里最geek的软件了，所以并不是推荐所有人都用它，不过AHK确实是你在Windows下能找到的最强大最实用的工具之一。你可以敲上几句代码就能把任何操作浓缩到一个热键中，你也可以用它写出自己的小程序来。乍一看可能很黄很暴力，但其实学起来容易得出乎你想象，所以倘如曾几何时你曾经幻想过“要是能够一键搞定，洒家这辈子也值了”，那么AHK就是那天降小裤裤……
